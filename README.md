# React-SPA Matador-Bull Arena
A small application to learn main concepts of functional and class React. 
Includes not stable Arena component and 2 moving objects: bull and matador. 

## Task
Matador is a memoized component, that rerenders on changes of its state or props. Business logic: matador moves depending on the bull position and plays sound of applause depending on passed props.
Matador.jsx - is a Functional-based React component; OldMatador.tsx - is a Class-based React component. 

## Run
1. `npm i` - install dependencies
2. `npm run dev` - run application on url http://localhost:5173/frontend-bull-arena/
3. more commands in the file package.json
